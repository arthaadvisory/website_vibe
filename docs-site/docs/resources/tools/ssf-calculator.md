# Social Security Fund (SSF) Calculator

*Estimate your monthly contribution liability under the Contribution-based Social Security Act.*

> *Note: This tool provides an estimate based on current contribution rates (31% of Basic Salary).*

## Contribution Breakdown

| Category | Contributor | Rate (%) | Allocation |
| :--- | :--- | :--- | :--- |
| **Provident Fund** | Employer (10%) + Employee (10%) | 20% | Retirement Savings |
| **Gratuity** | Employer | 8.33% | Termination Benefit |
| **Social Security Scheme** | Employer | 1.67% | Medical/Accident Insurance |
| **Unemployment Insurance** | Employee | 1% | Job Loss Support |
| **Total** | **Employer (20%) + Employee (11%)** | **31%** | |

### Formula for Manual Calculation
*   **Employer Pays:** `Basic Salary * 0.20`
*   **Employee Deducts:** `Basic Salary * 0.11`
*   **Total Deposit to SSF:** `Basic Salary * 0.31`
